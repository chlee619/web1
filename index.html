WEB

WWW는 World Wide Web이다. 간단하게 웹(Web)으로 부른다. 웹은 인터넷으로 접근할 수 있고 하이퍼텍스트로 연결되는 Uniform Resource Locators (URLs)으로 식별하는 문서나 자원이 있는 정보공간이다. 영국 과학자 Tim Berners-Lee가 1989년에 만들었다. 1990년 핵연구를 위한 유럽 조직인 CERN에서 일할 때 첫 웹브라우저를 선보였다. 1991년 웹브라우저를 조직 밖으로 공개하였다. 1991년 1월에 다른 연구 협회에 공개하였고 8월에 일반에 공개하였다. 월드 와이드 웹 (World Wide Web)은 정보화 시대가 발전하는데 핵심이었으며 인터넷을 통한 상호 작용에 수십억 명이 주요 도구로 사용하고 있다. 웹 페이지는 주로 HTML (Hypertext Markup Language)로 형식이 지정되고 주석이 달린 텍스트 문서이다. 형식이 지정된 텍스트 외에도 웹 페이지에는 이미지, 비디오, 오디오 및 소프트웨어와 같은 구성 요서가 포함될 수 있다. 구성 요소는 사용자의 웹 브라우저에서 멀티미디어 콘텐트의 일관된 페이지로 보여진다. 사용자는 문서에 들어있는 하이퍼 링크를 통해 웹 페이지 사이를 이동할 수 있다. 공통 주제와 공통 도메인 이름을 가진 여러 웹 페이지가 웹 사이트를 만든다. 웹 사이트 콘텐츠는 주로 게시자가 제공하거나 대화 형으로 제공한다. 사용자가 콘텐츠를 제공하거나 콘텐츠 사용자 또는 사용자의 작업에 따라 다르다. 웹 사이트는 놀이나 판매를 위해 쓰이거나 정부 또는 비정부 조직의 목적을 위해 쓰인다.
